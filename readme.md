## ⚙️ CLI Tools (Built in Go)

Terminal-first tooling for modern workflows.

- 🏠 **[homepodctl](https://github.com/agisilaos/homepodctl)**: macOS CLI that controls Apple Music playback and HomePod.

- 📅 **[acal](https://github.com/agisilaos/acal)**: Fast Apple Calendar CLI for terminal workflows and agent automation.

- 🎧 **[pocketcastsctl](https://github.com/agisilaos/pocketcastsctl)**: Control Pocket Casts directly from the terminal.

- 🔐 **[protonmailcli](https://github.com/agisilaos/protonmailcli)**: Minimal ProtonMail interaction via CLI.

- ✅ **[todoist-cli](https://github.com/agisilaos/todoist-cli)**: Manage Todoist tasks directly from the terminal.

- 🧾 **[notion-expenses-cli](https://github.com/agisilaos/notion-expenses-cli)**: Import transactions into Notion using a Go-based CLI.

Focused on:
- Clear UX in the terminal  
- Structured output (JSON-friendly)  
- Stable API integrations  
- Automation-ready workflows
- Agentic friendly
